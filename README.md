# 📊 Sales Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing sales data to uncover key business insights, trends, and performance metrics. The dashboard helps in understanding sales performance, profit distribution, and growth patterns using interactive visualizations.

---

## 🛠 Tools & Technologies
- Microsoft Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Power Query (Data Transformation)
- DAX (KPIs & Calculations)

---

## 📂 Dataset
- Sales dataset containing order details, customer information, product categories, and revenue data.

---

## 📊 Key Features
- Data cleaning and transformation using Power Query  
- Created KPIs such as Total Sales, Profit, and Growth Trends using DAX  
- Interactive dashboard with filters and slicers  
- Region-wise and category-wise sales analysis  
- Sales forecasting and trend analysis
